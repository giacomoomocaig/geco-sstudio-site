Specifiche tecniche

Come preparare e inviare i file

Per ottenere il miglior risultato possibile, è importante inviare materiale ordinato, ben esportato e accompagnato da indicazioni chiare.

Prima di inviare il materiale

Più il materiale arriva pulito e ben organizzato, più il lavoro può essere svolto in modo preciso, rapido ed efficace.

Se possibile, allega sempre anche un riferimento sonoro e una breve descrizione della direzione artistica che vuoi raggiungere.

Specifiche per il missaggio

  • I file devono avere una risoluzione minima di 44.1 kHz / 16 bit
  • I file devono avere una risoluzione massima di 96 kHz / 32 bit float
  • Ogni file non deve raggiungere 0 dBFS, per evitare clipping digitale
  • Ogni file deve avere un’etichettatura chiara e ordinata
  • Esempio: 1-Kick, 2-Snare, 3-Bass, 4-Guitar, 5-Keyboards
  • Se possibile, allega anche un rough mix realizzato da te
  • Allega una traccia di riferimento di un artista di cui apprezzi il suono
  • Se possibile, allega una breve descrizione del messaggio o delle emozioni che vuoi trasmettere

Specifiche per la masterizzazione

  • I file devono avere una risoluzione minima di 44.1 kHz / 16 bit
  • I file devono avere una risoluzione massima di 96 kHz / 32 bit float
  • I file non devono superare circa -5 / -6 dBFS
  • In caso di stem mastering, gli stems non possono superare le 6 unità
  • Gli stems devono avere un’etichettatura appropriata e chiara
  • Esempio: Gruppo Drums, Gruppo Background Vocals, Gruppo Chitarre
  • Allega un brano di riferimento per il suono desiderato
  • In caso di album o EP, allega la tracklist
  • Se vuoi inserire metadati nel brano, invia tutte le informazioni necessarie

Consegne e revisioni

In caso di masterizzazione, la consegna è generalmente di 5 giorni lavorativi.

In caso di missaggio, i tempi dipendono dal numero di tracce coinvolte e dalla complessità del progetto.

Le revisioni disponibili sono generalmente 3. Se sono necessarie ulteriori modifiche, possono essere concordate successivamente.

Materiale utile da allegare

  • Rough mix del brano, se disponibile
  • Uno o più riferimenti sonori
  • Tracklist, nel caso di EP o album
  • Note tecniche o artistiche utili
  • Indicazioni sui metadati, se richiesti

Hai dubbi prima dell’invio?

Se non sei sicuro di come esportare o organizzare il materiale, scrivici prima: ti aiuteremo a capire come prepararlo correttamente.
