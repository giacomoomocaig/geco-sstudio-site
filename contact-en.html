What to include in your request

  • Full name
  • Artist or project name
  • Contact email
  • Requested service: Mix / Stereo Master / Stem Master
  • Number of tracks
  • Number of channels, if it is a mix
  • Promo code, if available
  • Short description of the project and the result you are aiming for

Direct discussion

If needed, we are available for phone calls or video calls, so we can better understand the project, expectations and desired artistic direction.
